# Spring REST Hands-on

This project contains the Spring REST hands-on exercises completed as part of **Module 7 – Spring REST using Spring Boot 3** in the Cognizant Java Full Stack Engineer (FSE) Deepskilling Program.

---

## Project Overview

This project demonstrates the development of RESTful web services using Spring Boot.

## Hands-ons Included

### Employee REST API

- Employee Model
- Department Model
- Employee DAO
- Employee Service
- Employee Controller
- Exception Handling

### Country REST API

- Country Model
- Country DAO
- Country Service
- Country Controller
- REST Endpoints

---

## Technologies Used

- Java 17
- Spring Boot
- Spring Web
- Maven
- REST API

## How to Run

1. Open the project in IntelliJ IDEA.
2. Build the project using Maven.
3. Run the Spring Boot application.
4. Test the REST endpoints using Postman or a web browser.
